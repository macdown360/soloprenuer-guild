# soloprenuer-guild

## プロダクトコンセプト

ソロプレナー・ギルドは、一人だと挫折しやすいソロアントレプレナーを応援するためのプラットフォームです。

開発や事業活動を一人でも続けられるように、ランクアップやXPなどのゲーム的なインセンティブを与え、日々の行動を継続しやすくします。単なるコミュニティサイトではなく、ソロプレナーが「今日も少し前に進む」ための冒険者ギルド型サービスを目指します。

## 主要な価値

- 継続支援: ランクを上げる、XPを得る、クエストを達成するという仕組みで、一人でも開発や事業活動を続ける理由を作る。
- コミュニティ形成: 地域や専門領域ごとの小部屋を作り、情報交換、相談、仲間づくりができるようにする。
- 依頼と収益化: 依頼掲示板で得意な分野や困っていることを渡し合い、収益や事業機会につなげる。
- ナレッジ共有: 記録を共有することで、成功事例、失敗事例、おすすめツール、日々の学びをギルド全体の知識資産にする。

## 設計判断の軸

今後の文言、画面設計、機能追加では、以下を優先します。

- ソロプレナーが孤独に折れず、継続できること。
- ゲーム感覚で自然に行動したくなること。
- 地域や専門領域のつながりが、相談だけでなく事業機会にも結びつくこと。
- 個人の記録が、他の冒険者にとっても役立つナレッジになること。
- 世界観は「ソロプレナー向けの冒険者ギルド」として統一すること。
